# Description
The package contains 5 math games:
1. Even number
2. Calculator
3. Greatest Common Divisor
4. Arithmetic progression
5. Prime number
# Minimum Requirements
1. python 3.10
2. prompt 0.4.1
3. flake8 5.0.4
# Setup and run 'Even number' game
## Setup
`git clone git@github.com:MariaVPetrenko/python-project-49.git`
